# 1. Run Vault docker
{% highlight shell %}
docker run --cap-add=IPC_LOCK\
	-e 'VAULT_DEV_ROOT_TOKEN_ID=myroot'\
	-e 'VAULT_DEV_LISTEN_ADDRESS=0.0.0.0:8200'\
	-e 'VAULT_ADDR=http://127.0.0.1:8200'\
	-e 'VAULT_TOKEN=myroot'\
	-p 8200:8200\
	--name vaul-container\
    vault
{% endhighlight %}
Testing at <a href="http://localhost:8200/ui" target="_blank">http://localhost:8200/ui</a>

# 2. Run Postgres docker
{% highlight shell %}
docker run -d --name postgres\
    -p 5432:5432\
    -e POSTGRES_PASSWORD=postgres123456\
    -e POSTGRES_USER=postgres \
    postgres
{% endhighlight %}

## 3.2 Using CLI or HTTP request to apply config
Open Vault CLI tool in Web UI
![SignIn](/assets/media/cli-vault.png)

{% highlight shell %}
vault write database/config/postgres plugin_name=postgresql-database-plugin allowed_roles="default" connection_url="postgresql://{{username}}:{{password}}@IP_ADDRESS:5432?sslmode=disable" username="postgres" password="postgres123456"
{% endhighlight %}
Replacing `IP_ADDRESS` with your ip addresss

Or using cURL
{% highlight shell %}
curl --header "X-Vault-Token: myroot" --request POST --data '{"plugin_name": "postgresql-database-plugin","allowed_roles": "default","connection_url": "postgresql://{{username}}:{{password}}@localhost:5432?sslmode=disable","username": "postgres","password": "postgres123456"}' http://localhost:8200/v1/database/config/postgres
{% endhighlight %}

Grant privileges to role
{% highlight shell %}
vault write database/roles/default db_name=postgres creation_statements="CREATE ROLE \"{{name}}\" WITH LOGIN PASSWORD '{{password}}' VALID UNTIL '{{expiration}}';GRANT SELECT, UPDATE, INSERT ON ALL TABLES IN SCHEMA public TO \"{{name}}\";GRANT USAGE,  SELECT ON ALL SEQUENCES IN SCHEMA public TO \"{{name}}\";" default_ttl="1h" max_ttl="24h"
{% endhighlight %}

# 4. Springboot connect Postgres
## 4.1 Configuring in Spring
With Spring, we must configure in `bootstrap.yml` or `bootstrap.properties` file (not `application.yml` or `application.properties`)
{% highlight yaml %}
server:
  port: 8080
spring:
  application:
    name: callme-service
  cloud:
    vault:
      uri: http://localhost:8200
      token: 'myroot'
      postgresql:
        enabled: true
        role: default
        backend: database
  datasource:
    url: jdbc:postgresql://localhost:5432/postgres
  jpa.hibernate.ddl-auto: update
{% endhighlight %}
